Game Of Conway Java

El juego de la vida fue hecho por primera vez por Jhon Horton Conway en 1970.

Mi versión de este juego tiene un menú al iniciarse que te permite poner un nombre, poner un tamaño al mundo, cambiar las normas, colocar las células, jugar y salir.

AirControl Java

Un proyecto de clase que trata de hacer un juego sobre un control aéreo para practicar las clases y objetos.

Mi versión de este juego tiene un menú donde tienes las opciones:

  • Inicializar un avión aportando los valores que pide el programa
    • Matrícula
    • Autonomía
    • Altura
    • Capacidad de carga
  • Mover un avión creado
  • Revisar él espació aéreo
  • Salir
